# Orbit - SWECT Virtual Assistant

## Overview

Orbit is an AI-powered college information chatbot developed using Python, NLTK, and PyTorch. The chatbot uses Natural Language Processing (NLP) techniques and a neural network model to understand user queries and provide relevant responses.

The project is designed to assist students by answering common questions related to college information, departments, admissions, fees, and academic services.

## Features

* Intent-based chatbot
* Natural Language Processing using NLTK
* Tokenization and Stemming
* Bag of Words feature extraction
* Neural Network classification using PyTorch
* Interactive command-line interface
* Easy customization through intents.json

## Technologies Used

* Python
* PyTorch
* NLTK
* NumPy

## Installation

1. Clone the repository

git clone https://github.com/your-username/Orbit-SWECT-Virtual-Assistant.git

2. Install dependencies

pip install -r requirements.txt

3. Train the model

python train.py

4. Run the chatbot

python chatbot.py
